## Title of the Project
**A Comparative Evaluation of Classical and Transformer Based Architectures for Multi Class Cyberbullying Detection in Social Media**

## Selected Theme
**Multi Class Cyberbullying Classification**

## Brief Description
The project will addresses the critical challenge of identifying and categorizing multiple forms of online harassment, that includes ethnicity, religion, gender, and age-based bullying. 

Utilizing a balanced dataset of approximately 48,000 labeled tweets, we will follow the required methodological sequence to compare two distinct NLP generations.

## Stage 1: Classical Approach
First, we will establish a baseline using **Stage 1: Classical Approach**, employing **TF-IDF vectorization** paired with a **Logistic Regression** or **Naive Bayes** classifier.

## Stage 2: Recent Approach
After that stage, we will implement **Stage 2: Recent Approach** by fine tuning a transformer based model (e.g., **BERT** or **DistilBERT**) to capture deep semantic context and nuanced language patterns.

## Comparative Evaluation
The project will end with a comparative evaluation and critical analysis of model performance, computational trade offs, and ethical implications, supported by **t-SNE visualizations** of the learned semantic embeddings.
